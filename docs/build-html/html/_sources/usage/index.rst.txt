Compiler toolchains
=====================

For the usage of the toolchains, must to add environment variables
I list the nessesary ones.

+-------------------------------+------------------------------+--------------------------------------------------------------+
| Compiler toolchain file       | ENV_VARIABLE                 | Typical path                                                 |
+===============================+==============================+==============================================================+
| toolchain-mingw.cmake         | MINGW_TOOLCHAIN_PATH         | C:\MinGW\bin                                                 |
+-------------------------------+------------------------------+--------------------------------------------------------------+
| toolchain-ti-cl2000-c28.cmake | C28_TOOLCHAIN_PATH           | C:\ti\ccs1120\ccs\tools\compiler\ti-cgt-c2000_21.6.0.LTS\bin |
+-------------------------------+------------------------------+--------------------------------------------------------------+
| toolchain-ti-armcl.cmake      | M3_TOOLCHAIN_PATH            | C:\ti\ccs1120\ccs\tools\compiler\ti-cgt-arm_20.2.5.LTS\bin   |
+-------------------------------+------------------------------+--------------------------------------------------------------+
| toolchain-xc16.cmake          | XC16_TOOLCCHAIN_PATH         | C:\Microchip\xc16\v2.00\bin                                  |
+-------------------------------+------------------------------+--------------------------------------------------------------+
| toolchain-stm32l4.cmake       | ARM_NONE_EABI_TOOLCHAIN_PATH | C:\arm-none-eabi\11_3\bin                                    |
+-------------------------------+------------------------------+--------------------------------------------------------------+
