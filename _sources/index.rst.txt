The inspiration behind this project is to make easier to create and expand C/C++ projects build system.

It has a bunch of 
   - Helper functions
   - Output generators for microcontrollers
   - Compiler toolchain collection
   - IDE project generators (python based)

I used in it cmake version 3.18 restriction.
